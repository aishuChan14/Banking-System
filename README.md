# 🏦 Banking System in C++

This is a simple console-based banking system built using C++.

## 💡 Features
- Create a new account
- View account details
- Deposit money
- Withdraw money
- Delete account

## 📁 File Handling
Accounts are stored in a binary file `accounts.dat` for data persistence.

## 🛠 Technologies
- C++
- File I/O
- Object-Oriented Programming

## 🚀 How to Run
1. Compile: `g++ BankingSystem.cpp -o bank`
2. Run: `./bank`

## 📌 Note
Run in the same directory where `accounts.dat` will be saved.
